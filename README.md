# Terps-Raising-Pups-Site
A student built mock website for Terps Raising Pups at the University of Maryland, created with React and Vite.

## Running the Site Locally

Clone the repository:

```bash
git clone https://github.com/JacksonBennett05/Terps-Raising-Pups-Site.git
cd Terps-Raising-Pups-Site
```

Install Dependencies and run dev server
```bash
npm install
npm run dev
```
Then just open the local host URL displayed in the terminal
